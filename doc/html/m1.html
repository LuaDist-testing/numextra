reorder
Returns a new matrix that is a 'reordering' of the rows of matrix X. This reordering can be arbitrary, so the new matrix can also contain less rows or more rows than the original one. The new order, or composition, is given by the vector order. If the parameter col is true, the columns are reordered instead

  2. order
  3. col

Return

  1. A new matrix with the X rows (or columns) reordered

Broadcast function applier
Demonstrates the usage of the bsx function

Code

>  A = matrix{ {1, 2}, {3, 4}, {5, 6} }
>  b = matrix{ {3, 3} }
>  matrix.bsx(matrix.badd, A, b)
>  print( A )
   4   5
   6   7
   8   9
Filter
Filtering values from a vector with the filter function

Code

>  x = matrix{2,1,3,5,7,6,8}
>  f = x:filter(function(i) return x[i] % 2 == 0 end)
>  print( f )
   2   6   8
Power
Usage of the function mpow to perform exponentiation by squaring

Code

>  X = matrix{{1,2},{3,4}}
>  print( X:mpow(0) )
   1   0
   0   1
>  print( X:mpow(5) )
   1069   1558
   2337   3406
>  X = matrix{{0.3,0.7},{0.6,0.4}}
>  print( X:mpow(9999999) )
   0.461538   0.538462
   0.461538   0.538462
